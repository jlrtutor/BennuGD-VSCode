Build BennuGD2 Projects Faster

VS Code extension focused on BennuGD2 workflow: install, configure paths, compile, and compile+run with one shortcut. Includes v1 compatibility and LSP support based on Bennu source exports.

VS Code Extension BennuGD2 First LSP + Compile + Run Created in 2026

Activation Flow (Recommended)

1. Clone project: 
git clone https://github.com/jlrtutor/BennuGD-VSCode.git
cd BennuGD-VSCode
2. Build and package: 
npm install
npm run compile
npx @vscode/vsce package
3. In VS Code go to Extensions > ... > Install from VSIX... and select bennugd-vscode-support-1.0.1.vsix.
4. Open Settings and run BennuGD: Configure Paths....
5. Set Source root, Compiler path and Runtime path to BennuGD binary folders or exact binaries.
6. Open your BennuGD project and run BennuGD: Compile and Run Current File from the command palette.
7. Keyboard shortcuts: Ctrl+Alt+B / Cmd+Alt+B (compile), Ctrl+Alt+R / Cmd+Alt+R (compile + run).

What You Get

- BennuGD2 syntax by default
- BennuGD v1 language mode
- Hover, completion, definitions
- Diagnostics and compile output
- Configure panel with Browse buttons

Example Settings (BennuGD2, macOS arm64)

Apple Silicon binaries reference: github.com/jlrtutor/BennuGD2 

{
  "bennugd.defaultVersion": "v2",
  "bennugd.v2.sourceRoot": "<BENNUGD2_ROOT>",
  "bennugd.v2.compilerPath": "<BENNUGD2_ROOT>/build/macos-arm64/bin/bgdc",
  "bennugd.v2.runtimePath": "<BENNUGD2_ROOT>/build/macos-arm64/bin/bgdi"
}

Help: npm Not Installed

If you get npm: command not found, install Node.js LTS first (npm is included). Official download: nodejs.org/en/download. 

# Verify installation
node -v
npm -v

# macOS (Homebrew)
brew install node

# Windows (PowerShell + winget)
winget install OpenJS.NodeJS.LTS

# Ubuntu/Debian
sudo apt update
sudo apt install -y nodejs npm

# Fedora
sudo dnf install -y nodejs npm

# Arch
sudo pacman -S nodejs npm
